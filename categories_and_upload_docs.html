ThankU App: Categories & Upload Mechanics

1. Item Upload Procedure

Below is the step-by-step procedure of how an item is posted:

  1. Step 1: Category Selection: The user selects a single top-level category from the list.
  2. Step 2: Form Details:
    • Location is auto-detected via GPS.
    • Up to 4 images can be selected from the user's device.
    • Text details (Title, Description, Condition, Transportation) are filled out.
  3. Image Processing & Storage:

    The app processes the selected images and securely uploads them to the cloud storage system under a unique user folder.

  4. Record Creation:

    Once all images are successfully uploaded, a new public listing record is created in the main database containing all the item's details.

Information Saved for Each Item

  • Title: The name of the item.
  • Description: Additional details provided by the user.
  • Price: Always set to "Free".
  • Location: The city or neighborhood name.
  • GPS Coordinates: Exact latitude and longitude for mapping.
  • Category: The main classification of the item.
  • Condition: The physical state of the item (e.g., Good, New).
  • Transportation: How the item can be moved (e.g., fits in a car).
  • Images: The main cover photo and all additional photos.
  • User Details: A secure link to the profile of the person donating.
  • Expiry Time: Specifically for food items, indicating how long it is safe to consume.
